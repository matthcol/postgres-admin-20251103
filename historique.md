# Historique de PostgreSQL

- 1973-1985: Ingres by Michael Stonebraker (Berkeley University, California and commercial)
- 1986-1994: POSTGRES by Michael Stonebraker (Berkeley University, California)
    after developing Ingres (1973-1985)
- 1989: 1st official version
- 1995: Postgres95 - POSTQUEL => standard SQL (Andrew Yu and Jolly Chen)
- 1996: PostgreSQL, open source project, PostgreSQL Global Development Group (PGDG),
PostgreSQL.org
